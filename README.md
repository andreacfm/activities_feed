# CustomStreams

Create custom activities streams from a streamer "Model".
Just an idea up to now.

## Installation

Add this line to your application's Gemfile:

    gem 'custom_streams'

And then execute:

    $ bundle

Run the installer:

    rails g custom_streams:install

And then run the generated migration:

    rake db:migrate
